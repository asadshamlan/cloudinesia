Cloudinesia merupakan online portal pertama di Indonesia yang menyajikan berbagai informasi tentang Cloud Computing serta teknologi secara umum. Berangkat dari pengalaman berkutat di bidang penggunaan dan pengaplikasian Cloud Computing, kami memberikan media referensi kepada para pengguna cloud computing atau bagi mereka para praktisi IT maupun IT Eksekutif yang berminat mengeksplorasi bidang Cloud Computing.
